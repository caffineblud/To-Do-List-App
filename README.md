# To-Do List GUI App

***A simple and beginner-friendly To-Do List application built using Python and Tkinter.***

## Features:

- Add tasks
- Mark tasks as completed
- Delete tasks
- Persistent task storage using JSON
- Simple and clean GUI

## Technologies Used:

- Python
- Tkinter
- JSON
- File I/O

## Project Structure:
```bash
todo-list-gui/
│
├── main.py
├── tasks.json
├── requirements.txt
├── README.md
│
├── core/
│   └── task_manager.py
│
├── ui/
│   └── gui.py
│
├── utils/
│   └── file_handler.py
```
## Run the Project:
```python
python main.py
```
